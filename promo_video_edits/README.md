# Promo Video Agent - BlackOps

## Flujo local

1. Pon videos crudos en:
   `promo_video_edits/01_originales`

2. Abre en BlackOps:
   `/promo-video`

3. Escoge template, cantidad de clips, duracion y formato.

4. Los clips finales salen en:
   `promo_video_edits/03_listos_para_subir`

5. Cada corrida guarda un reporte en:
   `promo_video_edits/04_reportes`

## Templates iniciales

- Nightlife app: `BEST APP TO GO OUT`
- Promo dinners: `PROMO DINNERS TONIGHT`
- Pool party: `POOL PARTY THIS WEEK`
- Yacht party: `YACHT PARTY WEEKEND`
- Guestlist: `GUESTLIST OPEN`

## Notas

Esta primera version trabaja local con FFmpeg. Mas adelante podemos agregar descarga desde links, scoring de mejores momentos, aprobacion por cuenta, analytics por template y publicacion automatica.
